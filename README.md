# Family Cookbook 🍳

The family recipe collection — stored as Markdown, served as a PWA.

## Structure

```
recipes/
  chicken-parm.md
  beef-stew.md
  chocolate-cake.md
```

Flat directory. Each recipe is a Markdown file with YAML frontmatter — use `tags` for categorization (meal type, cuisine, etc.).
